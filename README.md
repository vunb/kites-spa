# Installation

Using npm / kites cli:

```bash
npm install -g @kites/cli
kites spa new-project
cd new-project
npm install
npm start
```

Using git clone:

```bash
git clone https://github.com/vunb/kites-spa.git new-project
cd new-project
npm install
npm start
```

Then open http://localhost:8889 on your favourite browser!
